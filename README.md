# Name to Marathi API

This is a simple REST API built using Node.js and Express.

## Features
- Convert English names to Marathi
- Supports GET and POST requests

## How to Run

npm install
node index.js

## API Endpoints

GET:
http://localhost:3000/convert/tushar

POST:
http://localhost:3000/convert
{
  "text": "tushar"
}
